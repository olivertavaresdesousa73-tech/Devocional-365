# 📸 Imagens do Site — Devocional 365

## Estrutura de arquivos esperada:

```
public/
  images/
    logo.png              ← Logo do Devocional 365
    caderno.png           ← Mockup do caderno impresso (hero)
    celular.png           ← Mockup do celular (hero)
    reviews/
      maria.jpg           ← Foto de Maria Santos
      ana.jpg             ← Foto de Ana Paula R.
      carlos.jpg          ← Foto de Carlos Eduardo
      juliana.jpg         ← Foto de Juliana Ferreira
```

---

## 📋 Especificações de cada imagem:

### logo.png
- **Onde aparece**: Cabeçalho, seção de oferta, rodapé
- **Tamanho recomendado**: 200x200px ou 300x100px
- **Formato**: PNG com fundo transparente
- **Dica**: Pode ser quadrado ou retangular

### caderno.png
- **Onde aparece**: Hero (início da página) — lado esquerdo
- **Tamanho recomendado**: 600x800px ou maior
- **Formato**: PNG com fundo transparente (recomendado)
- **Dica**: Use um mockup de caderno/livro aberto mostrando uma página do devocional

### celular.png
- **Onde aparece**: Hero (início da página) — lado direito
- **Tamanho recomendado**: 400x800px ou maior
- **Formato**: PNG com fundo transparente (recomendado)
- **Dica**: Use um mockup de celular mostrando o devocional na tela

### reviews/ (fotos de avaliação)
- **Onde aparecem**: Seção de avaliações + mini social proof na oferta
- **Tamanho recomendado**: 200x200px (quadrado)
- **Formato**: JPG ou PNG
- **Dica**: Rosto centralizado, boa iluminação, fundo neutro

---

## 🔄 Como atualizar:

1. Coloque os arquivos nesta pasta com os nomes exatos acima
2. O site carrega automaticamente — sem precisar editar código
3. Se uma imagem não existir, aparece um fallback (ícone ou iniciais)

- Os nomes dos arquivos devem ser EXATAMENTE como listados acima
- Use letras minúsculas
